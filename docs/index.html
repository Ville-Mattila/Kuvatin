Shrink images without the busywork.

A tiny native Windows app for batch image work — compression, format conversion, resizing and cropping — right from your Explorer right-click menu.

FEATURES

STOP 01

Compress PNG

The default. Lossy compression via libimagequant, finished with an oxipng pass — or go fully lossless. Big size drops, no visible loss.

libimagequant + oxipng
STOP 02

Convert

Move between PNG, JPEG, WebP, BMP, TIFF and GIF.

6 formats
STOP 03

Resize

By exact pixels, by percentage, or fit-to-box. Aspect-ratio aware with high-quality Lanczos3 resampling.

Lanczos3
STOP 04

Crop

Trim to a fixed size or aspect ratio with numeric width and height fields. Crop runs before resize for predictable output.

numeric fields
STOP 05

Batch in parallel

Drop a whole folder or a multi-selection. Kuvatin runs every file across all your cores and saves next to the originals — never overwriting.

rayon · multi-core
STOP 06

Right-click presets

Reusable presets live in the Windows Explorer menu. Select images, right-click, pick a preset — done, no window needed.

Explorer integration
